Pete Alta - Personal Website

Overview

This is my personal portfolio website, designed using HTML, CSS, and DaisyUI with the "forest" theme. It showcases my professional background, skills, and contact information.

Features

Modern, minimalistic design inspired by portfolio layouts.

Work experience section to highlight my IT support background.

Contact information with an email button for easy communication.

Links section for LinkedIn and Instagram.

Technologies Used

HTML

CSS

DaisyUI (Forest theme)
